# Polymath: First Principles Series

A minimalist, interactive collection of first-principles guides exploring the fundamental algorithms of the natural and digital worlds.

![Polymath Homepage Mockup](https://images.unsplash.com/photo-1451187580459-43490279c0fa?auto=format&fit=crop&q=80&w=1000)

## 🌐 Project Vision
The Polymath series is designed to decode complex systems—from the molecular machinery of life to the logic gates of artificial intelligence—stripping away layers of abstraction to reveal the core principles that govern our reality.

## 🚀 Key Features

- **Interactive Visualizations**: Every article features custom, high-performance Canvas-based simulations (e.g., Action Potentials, Neural Networks, Titan Knowledge Graphs).
- **Game of Life Background**: A live, interactive cellular automata simulation powers the homepage environment.
- **Retro Aesthetic**: Unified 8-bit typography and minimalist "glow" motion design for a focused, technical atmosphere.
- **Privacy First**: 100% self-hosted. All fonts and assets are served locally with zero external API dependencies or trackers.

## 🛠 Tech Stack

- **Core**: Vanilla HTML5, CSS3, and ES6+ JavaScript.
- **Graphics**: HTML5 Canvas API for all interactive simulations.
- **Typography**: Local 'Inter', 'Outfit', and 'Press Start 2P' assets.
- **Architecture**: No-build, zero-dependency static site for maximum portability and speed.

## 📂 Project Structure

- `index.html`: The central hub and portal to the series.
- `/html`: Comprehensive article pages covering 17+ different scientific and technical topics.
- `/css`: Modular stylesheets including a central `css/fonts.css` for local asset management.
- `/js`: Specialized visualization logic and background simulations.
- `/fonts`: Compressed local font binaries (WOFF2).

## 📥 How to Run

Since this is a vanilla web project, there is no installation required:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
